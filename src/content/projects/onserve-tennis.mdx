Website built in React for OnServe Tennis, including a refactored v2 (`OnServeTennis`, Aug 2022). Bootstrapped from Create React App, with component-based page structure, routing, and a content-driven layout for programs, schedules, and contact.

One of the earlier full-stack projects I built before turning primarily to ML research. Useful as a reference for the kind of product-facing work I can still ship when the occasion calls for it.
